# JPA 활용 1편 복습 & 활용 2편
인프런 강의 자바 ORM 표준 JPA 활용 1편 복습과 활용 2편을 
통해 JPA 코드 리팩토링 및 앞으로 들을 실전 Spring Data JPA, QueryDSL 
강의의 내용을 정리 실습내용들을 저장할 저장소입니다.

## 사용한 기술 & 개발 환경
- Java 11
- SpringBoot
- Spring Data JPA
- Thymeleaf
  - BootStrap
- Lombok
- DevTools
- H2 DataBase
  - 개발환경
